Summer 2022 Research Internship


Research Intern working with professor Jeffrey Herrmann on autonomous systems and metareasoning. Metareasoning is the process of reasoning about reasoning. It is used to monitor and control the decision making process and improve the efficiency and performance of algorithms.

The main task of the internship was to analyze the performance of two difference algorithms, LGR and MM regarding the perimeter defence problem. Machine learning models and exploratory data analysis were used to create a metareasoning policy. A paper will be published about the research using OverLeaf in the fall.

Project Tango Bitcamp Hackathon


Worked with a team of 4 members to create "Tango", a website focussed on connecting people with similar music interests. To gather information about music interests, Spotify's API was used. After collecting the information, the algorithm finds people with similar music interests that are also within a specific range in terms of distance. The query is passed through SQL and the result is shown visually on maps using Google's API . The database was maintained by CockroachDB and integrated with Python. The website was created by using HTML, CSS and flask. The code was shared and maintained using Github.
